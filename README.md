# oasys-genesis-contract

Genesis Smart Contracts for Oasys Blockchain.

# Setup(localhost)

## Install dependencies

```shell
npm install
```

## Build contracts

```shell
npm run build
```

## Build node
```shell
npx hardhat node
```

## Deploy contracts
```shell
npx hardhat --network localhost deploy:local
```

## Run tests

```shell
npm run test
```
